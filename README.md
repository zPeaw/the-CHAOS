CHAOS-Engine

Harmless visual chaos & prank simulator for Windows.

============================================================
========================== TÜRKÇE ==========================
============================================================

CHAOS-Engine, MEMZ tarzı kaotik demo yazılımlarından ilham alan fakat
GERÇEK ZARAR VERMEYEN bir görsel şaka motorudur.

Bilgisayarı bozmaz. Dosya silmez. Şifreleme yapmaz.

Sadece kontrollü, ayarlanabilir KAOS üretir.

  ------
  Amaç
  ------

Eğlence, yayın, arkadaş şakası veya görsel efekt denemeleri için:

-   sistemi korkutucu göstermek
-   kullanıcıyı panikletmek
-   ama sonunda hiçbir zarar bırakmamak

  ------------
  Özellikler
  ------------

-   Mouse titretme / zıplatma
-   Gerçek zamanlı ekran bozulmaları
-   Sahte uygulama hataları
-   Rastgele site & sekme açma
-   YouTube videoları
-   Garip Google aramaları
-   CMD pencereleri
-   Görev yöneticisini kapatma denemesi
-   Extreme Mode (çok daha agresif)

  -------------------
  GUI Yapılandırıcı
  -------------------

Program açıldığında modern bir panel gelir.

Buradan: - hangi efektlerin çalışacağını seçersin - süre belirlersin -
dil değiştirirsin - tema seçersin

Ayarlar otomatik kaydedilir.

  --------------
  Süre Sistemi
  --------------

Kaos sonsuza kadar sürmek zorunda değil. Süre bitince efektler durur ve
program kapanır.

  -------------
  Kill Switch
  -------------

ALT + 2 → program kapanır.

  -------------
  Güvenli mi?
  -------------

Evet.

Bu yazılım: - dosyaları değiştirmez - kayıt defterine bulaşmaz - kalıcı
değildir - veri toplamaz - internetten komut almaz

Kapatınca biter.

  --------------------------------
  Gerekli Kütüphaneler (Kurulum)
  --------------------------------

Minimum: pip install customtkinter

Sorun yaşarsan (önerilen): pip install –upgrade pip pip install
customtkinter

  ------------
  Çalıştırma
  ------------

python chaos_engine.py

  ------------------
  Sorumluluk Reddi
  ------------------

Bu proje eğlence amaçlıdır. Oluşabilecek korku, panik veya yanlış
anlaşılmalardan kullanıcı sorumludur.

============================================================
========================== ENGLISH =========================
============================================================

CHAOS-Engine is a visual prank & chaos simulator inspired by classic
chaotic demos but designed to be COMPLETELY HARMLESS.

It will NOT damage your system. It will NOT delete files. It will NOT
encrypt anything.

It only creates controlled, configurable CHAOS.

  ---------
  Purpose
  ---------

For fun, streams, demonstrations or experiments:

-   make the system look broken
-   create panic
-   leave zero real damage afterwards

  ----------
  Features
  ----------

-   Mouse shaking / jumping
-   Real-time screen distortions
-   Fake application errors
-   Random websites & tabs
-   YouTube videos
-   Weird Google searches
-   CMD windows
-   Attempts to block Task Manager
-   Extreme Mode (far more aggressive)

  ------------------
  GUI Configurator
  ------------------

At launch, a modern control panel appears.

From there you can: - choose which effects run - set duration - change
language - select theme

Settings are saved automatically.

  -----------------
  Duration System
  -----------------

Chaos doesn’t have to run forever.

When time is up: effects stop and the program exits.

  -------------
  Kill Switch
  -------------

ALT + 2 → instant exit.

  -------------
  Is it safe?
  -------------

Yes.

This software: - does not modify files - does not touch the registry -
is not persistent - does not collect data - does not receive remote
commands

Close it → it’s gone.

  -----------------------------------
  Required Libraries (Installation)
  -----------------------------------

Minimum: pip install customtkinter

If you run into issues (recommended): pip install –upgrade pip pip
install customtkinter

  -----
  Run
  -----

python chaos_engine.py
